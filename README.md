# üá∫üá∏


### üß∞ Take notes and record your thoughts directly from the terminal

Nerdnote allows you to create notes without leaving the terminal ‚Äî with a single command, your notes are saved in a predefined directory **(Documents/CNotes)** in **markdown (*.md)** format.

#### üìå Purpose

Often, you need to capture an idea or thought to avoid forgetting it. Usually, this requires picking up your smartphone or opening a separate note-taking app, which distracts you from your main work. Nerdnote is useful for those who spend most of their work in the terminal. It enables quick note-taking and continuation of work without interruptions.

---

#### üí° Features

* View notes (Preview). 
* List all notes. 
* Delete notes.
* Export to **pdf** or **html.**
* All operations are terminal-based, no need to leave it.

---

#### üìé Installation and Usage

The tool can be installed locally or globally. The difference is that with a local installation, it is available only within the virtual environment when active.

For installation click [here](/docs/install.md)

**Usage**

* `nn add note` ‚Äî creates an md file with the note, or appends to an existing one.
* `nn ls` ‚Äî shows all notes in the database with their ids.
* `nn cat id` ‚Äî displays the md file by id (id must be numeric).
* `nn rm id` ‚Äî deletes the file and record by id.
* `nn export id pdf` ‚Äî converts the file by id to pdf.
